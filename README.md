# Cyber Lab Hardware Proposal

This repository documents a technical hardware proposal developed for a cybersecurity lab environment, with a focus on performance, reliability, virtualization capacity, and budget control.

The project was designed as a structured procurement and technical justification case study for a lab scenario requiring multiple identical workstations for testing, simulation, and continuous technical use.

## Project Context

The proposal was developed for a cyber lab scenario that required:

- multiple homogeneous workstations
- support for virtualization and multitasking
- sustained operation in demanding technical environments
- controlled acquisition cost
- hardware reliability and maintainability
- room for future expansion

## Project Focus

This project covers the following areas:

- requirements analysis
- technical definition of workstation profile
- component selection and compatibility
- CPU, motherboard, RAM, storage, cooling, PSU, and case analysis
- virtualization-oriented sizing
- power planning
- stock validation
- budget calculation
- cost-per-unit analysis
- executive justification for decision makers

## Objectives

The main objective of this project was to define a coherent workstation proposal that could support a cybersecurity lab with:

- homogeneous equipment
- stable multitasking performance
- capacity for multiple virtual machines
- controlled thermal and electrical behavior
- realistic procurement feasibility
- financial compliance with a defined budget
